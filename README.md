# 🚀 API Health Monitoring Dashboard

## 📌 Overview
This project is a real-time API monitoring dashboard built using Django.  
It tracks the health and performance of multiple APIs and displays their status in a clean UI.

---

## ❗ Problem
In modern systems, multiple APIs are used for data flow.  
If an API fails or returns abnormal data, it can break the entire system.

---

## 💡 Solution
This dashboard automatically checks multiple APIs and shows:
- API working status (OK / ERROR)
- Record count
- System health

---

## ⚙️ Features
- ✅ Real-time API monitoring  
- 🔄 Auto refresh every 5 seconds  
- ❌ Error detection & simulation  
- 🤖 AI-based insights  
- 🚨 Alert system  
- 🎨 Clean UI  

---

## 🌐 APIs Used
- https://jsonplaceholder.typicode.com/posts  
- https://jsonplaceholder.typicode.com/users  

---

## 🛠️ Tech Stack
- Django  
- HTML, CSS, JavaScript  
- SQLite  

---
